# Social Media Analysis System

Prototype system for analyzing social media data, identifying trends, and studying user behavior patterns.

## Main components

- data loading and validation;
- text preprocessing;
- trend analysis;
- topic modeling;
- sentiment analysis;
- user activity analysis;
- PostgreSQL result storage;
- Streamlit interface;
- pytest tests.

## Run

- pip install -r requirements.txt
- python run_pipeline.py
- streamlit run app/streamlit_app.py
